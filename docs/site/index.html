EoStudio

Unified design suite with 12 editors: 3D modeler, CAD, image editor, game editor, UI/UX, UML, database designer, PCB editor, IDE. 30+ code generators and LLM AI.

Star on GitHub Quick Start

Features

Everything you need to design, model, and generate code — in one suite

12 Integrated Editors

Seamlessly switch between 3D, CAD, image, game, UI, UML, database, PCB, and code editors in a unified workspace.

3D Modeler & CAD Designer

Full parametric 3D modeling and CAD design with mesh editing, boolean operations, and export to STL/STEP.

Image/Game/UI Editor

Sprite editing, game level design, and UI/UX prototyping with drag-and-drop components and theming.

UML & Simulation Editor

Full UML diagramming with class, sequence, state machine, and activity diagrams plus simulation playback.

Database & PCB Designer

Visual database schema designer and PCB layout editor with DRC, auto-routing, and Gerber export.

30+ Code Generators

Generate production-ready code from diagrams and models for C, C++, Python, Rust, TypeScript, and more.

LLM-Powered AI Assistance

Integrated large language model AI for code completion, refactoring suggestions, and natural language queries.

WebSocket Real-Time Collaboration

Multi-user real-time collaboration via WebSocket with conflict resolution, cursors, and live preview sync.

Quick Start

Get up and running in seconds

npm install && npm run dev
